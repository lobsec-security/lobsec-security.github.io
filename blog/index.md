## Latest Posts

### [ClawHavoc: Uncovering 341 Malicious Skills on ClawHub](./clawhavoc-analysis.md)
*February 2025*

Our comprehensive analysis of the ClawHub skill registry revealed 341 malicious skills with a 17.4% malicious rate. We detail the attack vectors found—including prompt injection, data exfiltration, and wallet drains—and announce LobSec's response: an open-source skill scanner, verified registry, and AgentShield runtime protection.
